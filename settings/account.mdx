# Kontoeinstellungen

Die Kontoeinstellungen umfassen alle Präferenzen und Anpassungen für einen bestimmten Benutzer. Sie gelten nur für diesen spezifischen Benutzer und niemanden sonst. Die Kontoeinstellungen können von jedem Benutzer festgelegt werden.

## Profil

Im Profilbereich können Sie ein Bild hochladen und Ihren Namen, Ihre E-Mail-Adresse und Ihr Passwort ändern.
Um eine E-Mail-Adresse zu ändern, kontaktieren Sie den Support ([support@eraneos.com](mailto:support@eraneos.com)).

## Präferenzen

Abhängig von Ihrer Erfahrung und Ihren Vorlieben können Sie Ihr Konto so konfigurieren, dass es Ihren individuellen Bedürfnissen entspricht.

Sie können ein Standardmodell auswählen. Dieses Modell wird in jedem neuen Chat ausgewählt, es sei denn, Sie ändern es in der oberen linken Ecke.

Sie können Chat-Funktionen wie Websuche, Bilderzeugung oder Datenanalyse aktivieren oder deaktivieren. Ohne die Fähigkeit wird kein Chat in Ihrem Konto die entsprechende Funktion nutzen können.

Darunter können Sie die App-Sprache einstellen. Derzeit unterstützen wir Englisch und Deutsch.

## Benutzerdefinierte Anweisungen

Sie können in den Einstellungen Informationen über sich selbst oder darüber, wie die Antworten geschrieben werden sollen, hinzufügen. Dieser Text gilt für alle Nachrichten, die Sie an ein KI-Modell senden. Im Ressourcenbereich finden Sie einen Leitfaden mit Beispielen, was Sie in benutzerdefinierten Anweisungen hinzufügen können.

## Wissensordner

Sie können Dateien in einen Wissensordner einfügen und diesen Ordner einem Assistenten zuordnen. Dies ist hilfreich, um eine umfangreiche Dokumentation mit vielen Dateien an einen FAQ- oder Support-Assistenten anzuhängen. Diese Funktionalität funktioniert etwas anders als das direkte Anhängen von Dateien an einen Assistenten.

## Datenschutz

BusinessGPT verwendet telemetrische Daten, um das Produkt zu verbessern. In den Datenschutzeinstellungen können Sie die Telemetrie und Analytik deaktivieren.
