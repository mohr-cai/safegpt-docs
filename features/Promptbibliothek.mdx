Die Promptbibliotheken-Funktion in BusinessGPT ermöglicht es Ihnen, effektive Prompts (Anweisungen an die KI) zu erstellen, zu organisieren und mit Ihrem Team zu teilen. Promptbibliotheken dienen als Sammlungen bewährter Formulierungen, die konsistente und qualitativ hochwertige Interaktionen mit der KI sicherstellen.

## Was sind Promptbibliotheken?

Promptbibliotheken in BusinessGPT sind organisierte Sammlungen von vordefinierten Prompts, die für bestimmte Aufgaben oder Anwendungsfälle optimiert wurden. Sie ermöglichen es Teams, erfolgreiche Interaktionsmuster zu standardisieren und wiederzuverwenden, was die Effizienz und Konsistenz bei der Nutzung von BusinessGPT verbessert.

## Hauptfunktionen

### Prompt-Erstellung und -Organisation

- Erstellen Sie maßgeschneiderte Prompts für verschiedene Anwendungsfälle
- Organisieren Sie Prompts in thematischen Bibliotheken
- Kategorisieren und taggen Sie Prompts für einfache Auffindbarkeit
- Versionieren Sie Prompts, um Verbesserungen nachzuverfolgen

### Kollaboratives Teilen

- Teilen Sie Promptbibliotheken mit Teammitgliedern
- Festlegen von Zugriffsberechtigungen für verschiedene Benutzer über das Rechte-Rollen-Konzept
- Als Administrator können Sie Prompts unternehmensweit öffentlich machen
- Fördern Sie den Austausch von Best Practices im gesamten Unternehmen

### Prompt-Optimierung

- Testen Sie verschiedene Prompt-Varianten
- Bewerten Sie die Effektivität von Prompts
- Iterieren Sie Prompts basierend auf Feedback und Ergebnissen
- Dokumentieren Sie erfolgreiche Prompt-Strategien

## So verwenden Sie Promptbibliotheken

### Erstellen einer neuen Promptbibliothek

1. Navigieren Sie zum Bereich "Promptbibliotheken" im Seitenmenü
2. Klicken Sie auf das "+" Symbol, um eine neue Bibliothek zu erstellen
3. Geben Sie einen aussagekräftigen Namen im Feld "Bibliothekstitel" ein
4. Optional können Sie eine Beschreibung im Feld "Beschreibung" hinzufügen
5. Klicken Sie auf "Erstellen", um Ihre neue Promptbibliothek anzulegen

### Hinzufügen von Prompts zu einer Bibliothek

1. Öffnen Sie Ihre Promptbibliothek aus der Übersicht
2. Klicken Sie auf "Neuer Prompt", um einen Prompt hinzuzufügen
3. Geben Sie einen Titel und den Prompt-Text ein
4. Fügen Sie optional Tags und Kategorien hinzu
5. Speichern Sie den Prompt in Ihrer Bibliothek

### Teilen einer Promptbibliothek

1. Klicken Sie auf der Seite der Promptbibliothek auf "Zugriff verwalten"
2. Geben Sie die Namen oder E-Mail-Adressen der Kollegen ein, die Zugriff haben sollen
3. Legen Sie für jeden Benutzer geeignete Berechtigungen über das Rechte-Rollen-Konzept fest
4. Speichern Sie Ihre Freigabeeinstellungen

### Öffentliche Prompts (nur für Administratoren)

1. Öffnen Sie als Administrator die Promptbibliothek
2. Wählen Sie den Prompt aus, den Sie unternehmensweit teilen möchten
3. Klicken Sie auf "Öffentlich machen"
4. Der Prompt ist nun für alle Mitarbeiter im Unternehmen verfügbar

### Verwendung von Prompts in Chats

#### Über das Menü

1. Öffnen Sie einen bestehenden Chat oder starten Sie einen neuen Chat
2. Klicken Sie auf das Prompt-Menü-Symbol
3. Wählen Sie einen Prompt aus Ihren verfügbaren Bibliotheken
4. Der ausgewählte Prompt wird automatisch in den Chat eingefügt
5. Sie können den Prompt vor dem Senden noch anpassen

#### Mit Slashbefehlen (Schnellzugriff)

1. Geben Sie in der Chat-Eingabe `/` ein, um alle verfügbaren Prompts anzuzeigen
2. Oder geben Sie `/[Suchbegriff]` ein, um nach bestimmten Prompts zu suchen (z.B. `/email`)
3. Wählen Sie aus der angezeigten Liste den gewünschten Prompt aus
4. Der ausgewählte Prompt wird automatisch in den Chat eingefügt
5. Sie können den Prompt vor dem Senden noch anpassen

Diese Slashbefehle bieten einen schnellen und effizienten Zugriff auf Ihre Promptbibliotheken direkt während der Eingabe, ohne den Gesprächsfluss zu unterbrechen.

## Bewährte Praktiken

Für optimale Ergebnisse mit Promptbibliotheken:

1. **Erstellen Sie klare, spezifische Prompts** für eindeutige Anwendungsfälle
2. **Dokumentieren Sie den Zweck jedes Prompts** in seiner Beschreibung
3. **Kategorisieren Sie Prompts sorgfältig** für einfache Auffindbarkeit
4. **Iterieren und verbessern Sie Prompts** basierend auf den Ergebnissen
5. **Teilen Sie erfolgreiche Prompts** mit relevanten Teammitgliedern
6. **Standardisieren Sie Prompts für wiederkehrende Aufgaben** im gesamten Unternehmen
