Die Projekte-Funktion in BusinessGPT ermöglicht es Ihnen, Sammlungen von Dokumenten als thematische Wissensbasen zu erstellen, zu organisieren und zu teilen. Projekte dienen als Repositories für themenspezifische Inhalte, die innerhalb eines Teams analysiert, durchsucht oder geteilt werden können.

## Was sind Projekte?

Projekte in BusinessGPT sind organisierte Sammlungen von Dokumenten wie Word-, PDF- oder PowerPoint-Dateien, die eine themenbasierte Wissensdatenbank bilden. Sie ermöglichen es Abteilungen, ihr Fachwissen zu konsolidieren und für andere Mitarbeiter zugänglich zu machen, was die Zusammenarbeit und Effizienz verbessert.

## Hauptfunktionen

### Dokumentensammlung und -organisation

- Hochladen verschiedener Dokumenttypen (PDF, Word, PowerPoint, etc.)
- Strukturierte Organisation von Dokumenten
- Erstellung themenspezifischer Wissensrepositorien
- Versionskontrolle von Dokumenten

### Kollaboratives Wissensteilen

- Teilen von Projekten mit Teammitgliedern
- Festlegen von Zugriffsberechtigungen für verschiedene Benutzer
- Ermöglichung des abteilungsübergreifenden Wissensaustauschs
- Förderung des kollaborativen Lernens und der Forschung

### Integrierte Analyse

- Direkte Analyse hochgeladener Dokumente mit BusinessGPT
- Erstellung von Zusammenfassungen von Dokumentensammlungen
- Stellen spezifischer Fragen zum Inhalt
- Erhalt von Antworten mit Quellenangaben

## So verwenden Sie Projekte

### Erstellen eines neuen Projekts

1. Navigieren Sie zum Bereich "Projekte" im Seitenmenü
2. Klicken Sie auf das "+" Symbol, um ein neues Projekt zu erstellen
3. Geben Sie einen aussagekräftigen Namen im Feld "Projekttitel" ein
4. Optional können Sie eine Beschreibung im Feld "Projektbeschreibung" hinzufügen
5. Klicken Sie auf "Erstellen", um Ihr neues Projekt anzulegen

### Hinzufügen von Dokumenten zu einem Projekt

1. Öffnen Sie Ihr Projekt aus der Projektübersicht
2. Ziehen Sie Dokumente per Drag & Drop in den dafür vorgesehenen Bereich
3. Unterstützte Formate umfassen Word, PDF, PowerPoint und andere
4. Die hochgeladenen Dokumente bilden die Wissensbasis für dieses Projekt

### Teilen eines Projekts

1. Klicken Sie auf der Projektseite auf "Zugriff verwalten"
2. Geben Sie die Namen oder E-Mail-Adressen der Kollegen ein, die Zugriff haben sollen
3. Legen Sie für jeden Benutzer geeignete Berechtigungen fest
4. Speichern Sie Ihre Freigabeeinstellungen

### Verwendung von Projektwissen in Chats

1. Öffnen Sie einen bestehenden Chat oder starten Sie einen neuen Chat
2. Klicken Sie auf das "+" Symbol, um das Dropdown-Menü aufzurufen
3. Wählen Sie Ihr zuvor erstelltes Projekt aus
4. Klicken Sie auf "Speichern", um den Projektkontext auf Ihren Chat anzuwenden
5. BusinessGPT wird nun die Dokumente in diesem Projekt als Referenzmaterial verwenden

## Bewährte Praktiken

Für optimale Ergebnisse mit Projekten:

1. **Erstellen Sie fokussierte Projekte** für bestimmte Themen oder Bereiche
2. **Verwenden Sie klare Namenskonventionen** für Projekte und Dokumente
3. **Integrieren Sie verschiedene Dokumenttypen** für umfassendes Wissen
4. **Aktualisieren Sie Projektinhalte regelmäßig**, um die Relevanz zu erhalten
5. **Teilen Sie Projekte strategisch** mit relevanten Teammitgliedern
