## Grundlagen

Effektive Prompts sind der Schlüssel zu besseren Ergebnissen mit BusinessGPT. Ein gut strukturierter Prompt führt zu präziseren und nützlicheren Antworten.

## Struktur eines guten Prompts

1. **Klare Rolle definieren**: "Du bist ein erfahrener Marketingexperte..."
2. **Kontext bereitstellen**: Relevante Hintergrundinformationen angeben
3. **Spezifische Aufgabe**: Präzise formulieren, was gewünscht ist
4. **Format angeben**: Gewünschte Struktur der Antwort spezifizieren

## Bewährte Techniken

### Häufige Fehler vermeiden

- **❌ Zu vage**: "Schreibe etwas über Marketing"
- **✅ Spezifisch**: "Erstelle eine 5-Punkte-Strategie für Social Media Marketing"

- **❌ Fehlender Kontext**: "Wie kann ich das verbessern?"
- **✅ Mit Kontext**: "Wie kann ich diese E-Mail-Kampagne verbessern? [Text einfügen]"

## BusinessGPT-spezifische Tipps

### Wissensdatenbanken nutzen
```
Basierend auf dem hochgeladenen Handbuch, erkläre die wichtigsten Funktionen.
```

### Modellauswahl
- **GPT-4o**: Komplexe Analysen und kreative Aufgaben
- **Claude**: Nuancierte Textverarbeitung und längere Dokumente
- **Schnellere Modelle**: Einfache Fragen und Zusammenfassungen

### Slashbefehle
Nutzen Sie `/[Suchbegriff]` für schnellen Zugriff auf Prompts.

## Prompt-Templates

### Analyse-Template
```
Rolle: [Expertenrolle]
Kontext: [Hintergrundinformationen]
Aufgabe: Analysiere [Element]
Format: [gewünschte Struktur]
```

### Erstellungs-Template
```
Erstelle [Inhaltsart] für [Zielgruppe]
Stil: [Ton/Stil]
Länge: [Wortanzahl]
Struktur: [Gliederung]
```

## Best Practices

- **Persona-basiert**: Detaillierte Rollen für konsistente Ergebnisse
- **Constraint-basiert**: Bewusste Grenzen für fokussierte Antworten
- **Multi-Step**: Komplexe Aufgaben in Schritte aufteilen
- **Iterativ**: Prompts basierend auf Ergebnissen verfeinern
