Die Wissensdatenbank-Funktion in BusinessGPT ermöglicht es Ihnen, Sammlungen von Dokumenten als thematische Wissensbasen zu erstellen, zu organisieren und zu teilen. Wissensdatenbanken dienen als Repositories für themenspezifische Inhalte, die innerhalb eines Teams analysiert, durchsucht oder geteilt werden können.

## Was ist eine Wissensdatenbank?

Wissensdatenbanken in BusinessGPT sind organisierte Sammlungen von Dokumenten wie Word-, PDF- oder PowerPoint-Dateien, die eine themenbasierte Wissensbasis bilden. Sie ermöglichen es Abteilungen, ihr Fachwissen zu konsolidieren und für andere Mitarbeiter zugänglich zu machen, was die Zusammenarbeit und Effizienz verbessert.

## Hauptfunktionen

### Dokumentensammlung und -organisation

- Hochladen verschiedener Dokumenttypen (PDF, Word, PowerPoint, xlsx)
- Strukturierte Organisation von Dokumenten in Listen und Unterlisten
- Erstellung themenspezifischer Wissensrepositorien
- Versionskontrolle von Dokumenten

### Kollaboratives Wissensteilen

- Teilen von Wissensdatenbanken mit Teammitgliedern
- Festlegen von Zugriffsberechtigungen für verschiedene Benutzer über das Rechte-Rollen-Konzept
- Ermöglichung des abteilungsübergreifenden Wissensaustauschs
- Förderung des kollaborativen Lernens und der Forschung

### Integrierte Analyse

- Direkte Analyse hochgeladener Dokumente mit BusinessGPT
- Erstellung von Zusammenfassungen von Dokumentensammlungen
- Stellen spezifischer Fragen zum Inhalt
- Erhalt von Antworten mit Quellenangaben

## So verwenden Sie die Wissensdatenbank

### Erstellen einer neuen Wissensdatenbank

1. Navigieren Sie zum Bereich "Wissensdatenbank" im Seitenmenü
2. Klicken Sie auf das "+" Symbol, um eine neue Wissensdatenbank zu erstellen
3. Geben Sie einen aussagekräftigen Namen im Feld "Titel" ein
4. Optional können Sie eine Beschreibung im Feld "Beschreibung" hinzufügen
5. Klicken Sie auf "Erstellen", um Ihre neue Wissensdatenbank anzulegen

### Hinzufügen von Dokumenten zu einer Wissensdatenbank

1. Öffnen Sie Ihre Wissensdatenbank aus der Übersicht
2. Ziehen Sie Dokumente per Drag & Drop in den dafür vorgesehenen Bereich
3. Unterstützte Formate umfassen Word, PDF, PowerPoint und andere
4. Die hochgeladenen Dokumente bilden die Wissensbasis für diese Sammlung
5. Organisieren Sie Dokumente in Listen und Unterlisten für eine bessere Struktur

### Teilen einer Wissensdatenbank

1. Klicken Sie auf der Seite der Wissensdatenbank auf "Zugriff verwalten"
2. Geben Sie die Namen oder E-Mail-Adressen der Kollegen ein, die Zugriff haben sollen
3. Legen Sie für jeden Benutzer geeignete Berechtigungen über das Rechte-Rollen-Konzept fest
4. Speichern Sie Ihre Freigabeeinstellungen

### Verwendung von Wissen in Chats

#### Über das Menü

1. Öffnen Sie einen bestehenden Chat oder starten Sie einen neuen Chat
2. Klicken Sie auf das "+" Symbol, um das Dropdown-Menü aufzurufen
3. Wählen Sie Ihre zuvor erstellte Wissensdatenbank aus
4. Klicken Sie auf "Speichern", um den Kontext auf Ihren Chat anzuwenden
5. BusinessGPT wird nun die Dokumente in dieser Wissensdatenbank als Referenzmaterial verwenden

## Bewährte Praktiken

Für optimale Ergebnisse mit Wissensdatenbanken:

1. **Erstellen Sie fokussierte Sammlungen** für bestimmte Themen oder Bereiche
2. **Verwenden Sie klare Namenskonventionen** für Wissensdatenbanken und Dokumente
3. **Integrieren Sie verschiedene Dokumenttypen** für umfassendes Wissen
4. **Strukturieren Sie Ihre Dokumente in Listen und Unterlisten** für bessere Organisation
5. **Aktualisieren Sie Inhalte regelmäßig**, um die Relevanz zu erhalten
6. **Teilen Sie Wissensdatenbanken strategisch** mit relevanten Teammitgliedern
