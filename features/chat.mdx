Die BusinessGPT Chat-Oberfläche ermöglicht die Kommunikation mit der KI in natürlicher Sprache, etwa zum Beantworten von Fragen, Abrufen von Geschäftsdaten und Ausführen repetitiver Aufgaben.

## Nutzung

Die Chat-Oberfläche ist der zentrale Zugang zur KI von BusinessGPT

/>

## Funktionen

- **Interaktion in natürlicher Sprache:** Freie Formulierung von Fragen und Aufgaben.
- **Kontextbezug:** Erinnerung an vorherige Chat-Nachrichten im Gesprächsverlauf.
- **Integration von Geschäftsdaten:** Hochladen und Einbinden von Dokumenten als Wissensbasis für gezielte Fragen (z. B. "Fasse den Marketingbericht zusammen.").
- **Multimodal:** Verarbeitung von Texten, Tabellen, Grafiken, Dokumentzusammenfassungen.
- **Modellauswahl:** Wechsel zwischen unterschiedlichen KI-Modellen, je nach Anwendungsbedarf.
- **Slash-Befehle:** Schneller Zugriff auf gespeicherte Prompts und Funktionen über z.B. `/marketing`.

## Kurzanleitung Slash-Befehle

1. Geben Sie `/` ein und wählen Sie einen Prompt
2. Optional: Suchbegriff zum Filtern direkt im Anschluss an den Slash ergänzen
3. Enter drücken
4. Prompt wird direkt in den Chat eingefügt

## Empfehlungen für effektive Nutzung

1. Formulieren Sie klare, spezifische Fragen.
2. Kontext/Details angeben, falls benötigt.
3. Nutzen Sie Folgefragen für Details.
4. Prüfen Sie geschäftskritische Informationen, bevor Sie diese übernehmen.

## Einschränkungen

- Zugriff nur auf internes Wissen nur mittels aktiv bereitgestellte Dokumente/Wissensdaten.
- Eingeschränkte Kenntnis zu sehr aktuellen Ereignissen – wahlweise Websuche hinzuschalten.
- Für spezielle Branchenbegriffe ggf. zusätzlichen Kontext geben.
- Komplexe/zweideutige Fragen ggf. vermeiden bzw. nachschärfen .
