Die Websuche-Funktion von BusinessGPT ermöglicht der KI, im Internet nach relevanten Quellen zu suchen, die bei der Beantwortung Ihrer Fragen helfen können. Wenn aktiviert, erlaubt diese Funktion BusinessGPT, sein Wissen mit aktuellen Informationen aus dem Web zu ergänzen, was besonders nützlich für Anfragen zu aktuellen Ereignissen, spezifischen Fakten oder spezialisierten Themen ist.

## Hauptfunktionen

### Echtzeit-Informationszugriff

- Zugriff auf aktuelle Informationen aus dem Internet
- Ergänzung des BusinessGPT-Wissens mit externen Quellen
- Erhalten Sie Antworten auf Fragen zu aktuellen Ereignissen oder Entwicklungen
- Überprüfen Sie Fakten mit aktuellen Online-Quellen

### Quellenangabe

- Erhalten Sie Antworten mit Verweisen auf die verwendeten Webquellen
- Sehen Sie die URLs der Quellen zur Überprüfung
- Verstehen Sie, welche Teile der Antwort aus Webinhalten abgeleitet sind
- Überprüfen Sie Quellen für kritische Informationen manuell

### Selektive Aktivierung

- Aktivieren oder deaktivieren Sie die Websuche für jede Chat-Sitzung
- Kontrollieren Sie, wann auf externe Informationen zugegriffen wird
- Schalten Sie die Funktion mit einem einfachen Klick ein oder aus
- Wahren Sie die Privatsphäre, indem Sie den Webzugriff bei Nichtbedarf einschränken

## Verwendung der Websuche

### Aktivieren der Websuche

1. Öffnen Sie einen bestehenden Chat oder starten Sie einen neuen Chat
2. Suchen Sie die Option "Websuche umschalten" unter dem Chat-Eingabefeld
3. Klicken Sie, um die Websuche für die aktuelle Chat-Sitzung zu aktivieren
4. Die Funktion bleibt aktiv, bis Sie sie deaktivieren oder die Sitzung beenden

### Verwendung der Websuche in Gesprächen

Sobald die Websuche aktiviert ist:

1. Stellen Sie Fragen wie gewohnt
2. BusinessGPT durchsucht bei Bedarf automatisch das Web, um bessere Antworten zu liefern
3. Antworten enthalten Informationen sowohl aus dem Wissen von BusinessGPT als auch aus Webquellen
4. Quellenangaben werden bereitgestellt, wenn Webinhalte verwendet werden

## Best Practices

Für optimale Ergebnisse mit der Websuche:

1. **Seien Sie spezifisch in Ihren Anfragen**, um BusinessGPT zu helfen, die relevantesten Informationen zu finden
2. **Überprüfen Sie wichtige Informationen**, indem Sie die bereitgestellten Quellenlinks kontrollieren
3. **Nutzen Sie die Websuche selektiv** für Themen, die von aktuellen Informationen profitieren
4. **Kombinieren Sie mit anderen Funktionen** wie Wissensdatenbanken für umfassende Antworten
5. **Berücksichtigen Sie Datenschutzaspekte** bei der Aktivierung der Websuche für sensible Themen

## Einschränkungen

Aktuelle Einschränkungen der Websuche-Funktion:

- Suchergebnisse hängen von der Verfügbarkeit und Qualität der Online-Informationen ab
- Nicht alle Webinhalte sind zugänglich oder indizierbar
- Sehr aktuelle Ereignisse (innerhalb von Stunden) sind möglicherweise noch nicht gut online dokumentiert
- Quellen sollten für kritische Geschäftsentscheidungen manuell überprüft werden
