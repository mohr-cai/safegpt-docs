Die Assistenten-Funktion in BusinessGPT ermöglicht es Ihnen, spezialisierte KI-Konfigurationen zu erstellen, die auf bestimmte Aufgaben oder Anwendungsfälle zugeschnitten sind. Ähnlich wie bei spezialisierten GPTs können Sie Assistenten mit verschiedenen Wissensdatenbanken, Systemprompts und vordefinierten Funktionen ausstatten, um maßgeschneiderte KI-Erfahrungen zu schaffen.

## Was sind Assistenten?

Assistenten in BusinessGPT sind konfigurierbare KI-Profile, die für spezifische Anwendungsfälle optimiert wurden. Sie kombinieren verschiedene Komponenten wie Wissensdatenbanken, Systemprompts und Funktionen, um eine spezialisierte KI zu schaffen, die genau auf Ihre Bedürfnisse zugeschnitten ist.

## Hauptfunktionen

### Anpassbare Konfiguration

- Erstellen Sie maßgeschneiderte Assistenten für verschiedene Geschäftsbereiche
- Wählen Sie das optimale KI-Modell für Ihren spezifischen Anwendungsfall
- Kombinieren Sie mehrere Wissensdatenbanken für umfassende Informationsquellen
- Definieren Sie spezifische Systemprompts für konsistente Interaktionen
- Aktivieren Sie vordefinierte Funktionen wie Gedächtnis oder Websuche
- Speichern Sie Konfigurationen zur Wiederverwendung und zum Teilen

### Integrierte Komponenten

- **Wissensdatenbanken**: Binden Sie eine oder mehrere [Wissensdatenbanken](/features/knowledge-library) ein, um dem Assistenten Zugriff auf spezifische Dokumentsammlungen zu geben
- **Systemprompts**: Definieren Sie Prompts, die das Verhalten und die Antworten des Assistenten steuern
- **Gedächtnisfunktion**: Aktivieren Sie die [Gedächtnisfunktion](/features/memory), damit der Assistent Gesprächsverläufe berücksichtigen kann
- **Websuche**: Integrieren Sie die [Websuche](/features/web-search), um dem Assistenten Zugriff auf aktuelle Online-Informationen zu ermöglichen

### Kollaboratives Teilen

- Teilen Sie Assistenten mit Teammitgliedern
- Festlegen von Zugriffsberechtigungen für verschiedene Benutzer über das Rechte-Rollen-Konzept
- Standardisieren Sie KI-Interaktionen innerhalb von Teams oder Abteilungen
- Fördern Sie Best Practices durch gemeinsam genutzte Assistenten-Konfigurationen

## So verwenden Sie Assistenten

### Erstellen eines neuen Assistenten

1. Navigieren Sie zum Bereich "Assistenten" im Seitenmenü
2. Klicken Sie auf das "+" Symbol, um einen neuen Assistenten zu erstellen
3. Geben Sie einen aussagekräftigen Namen im Feld "Assistentenname" ein
4. Optional können Sie eine Beschreibung im Feld "Beschreibung" hinzufügen
5. Konfigurieren Sie die Komponenten Ihres Assistenten (siehe unten)
6. Klicken Sie auf "Erstellen", um Ihren neuen Assistenten zu speichern

### Konfigurieren der Assistenten-Komponenten

1. **Modell auswählen**:
   - Wählen Sie das optimale KI-Modell für Ihren Anwendungsfall aus der Dropdown-Liste
   - Verschiedene Modelle bieten unterschiedliche Stärken (z.B. GPT-4o für komplexe Aufgaben, Claude für nuancierte Textverarbeitung)
   - Das ausgewählte Modell bestimmt die Leistungscharakteristika Ihres Assistenten

2. **Wissensdatenbanken hinzufügen**:
   - Wählen Sie eine oder mehrere Wissensdatenbanken aus der Dropdown-Liste aus
   - Die ausgewählten Wissensdatenbanken werden als Wissensquellen für den Assistenten verwendet

3. **Systemprompts definieren**:
   - Geben Sie spezifische Prompts ein, die das Verhalten des Assistenten steuern
   - Definieren Sie die Rolle, den Ton und den Fokus des Assistenten
   - Legen Sie fest, wie der Assistent auf bestimmte Anfragen reagieren soll

4. **Funktionen aktivieren**:
   - Aktivieren oder deaktivieren Sie die Gedächtnisfunktion
   - Aktivieren oder deaktivieren Sie die Websuche
   - Konfigurieren Sie weitere verfügbare Funktionen nach Bedarf

### Verwenden eines Assistenten

1. Navigieren Sie zum Bereich "Assistenten" im Seitenmenü
2. Wählen Sie den gewünschten Assistenten aus der Liste aus
3. Klicken Sie auf "Starten", um eine neue Chat-Sitzung mit diesem Assistenten zu beginnen
4. Interagieren Sie mit dem Assistenten, der nun gemäß Ihrer Konfiguration arbeitet

### Assistenten anheften für direkten Zugriff

Sie können häufig verwendete Assistenten anheften, um schnell auf sie zuzugreifen:

1. **Assistent anheften**:
   - Klicken Sie auf das Pin-Symbol neben dem Assistenten in der Assistenten-Liste
   - Angeheftete Assistenten erscheinen in der Chat-Oberfläche für direkten Zugriff

2. **Direkter Zugriff im Chat**:
   - Angeheftete Assistenten sind direkt in der Chat-Oberfläche verfügbar
   - Klicken Sie auf den gewünschten Assistenten, um sofort eine neue Sitzung zu starten
   - Dies ermöglicht einen schnellen Wechsel zwischen verschiedenen spezialisierten Assistenten

### Teilen eines Assistenten

1. Öffnen Sie den Assistenten, den Sie teilen möchten
2. Klicken Sie auf "Zugriff verwalten"
3. Geben Sie die Namen oder E-Mail-Adressen der Kollegen ein, die Zugriff haben sollen
4. Legen Sie für jeden Benutzer geeignete Berechtigungen über das Rechte-Rollen-Konzept fest
5. Speichern Sie Ihre Freigabeeinstellungen

## Bewährte Praktiken

Für optimale Ergebnisse mit Assistenten:

1. **Erstellen Sie fokussierte Assistenten** für spezifische Anwendungsfälle oder Abteilungen
2. **Kombinieren Sie relevante Wissensdatenbanken** für umfassende Informationsquellen
3. **Formulieren Sie klare Systemprompts**, die das gewünschte Verhalten definieren
4. **Aktivieren Sie nur die benötigten Funktionen**, um die Effizienz zu maximieren
5. **Testen Sie Ihre Assistenten** mit typischen Anfragen, bevor Sie sie teilen
6. **Aktualisieren Sie Assistenten regelmäßig**, wenn sich Anforderungen ändern
