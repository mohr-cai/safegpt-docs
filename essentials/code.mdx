## Grundlagen

### Inline-Code

Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, schließen Sie es in Backticks (`) ein.

```
Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, schließen Sie es in Backticks (`) ein.
```

### Code-Block

Verwenden Sie [umzäunte Code-Blöcke](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), indem Sie Code in drei Backticks einschließen und den führenden Backticks die Programmiersprache Ihres Snippets folgen lassen, um Syntaxhervorhebung zu erhalten. Optional können Sie auch den Namen Ihres Codes nach der Programmiersprache schreiben.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
